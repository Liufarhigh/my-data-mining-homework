- 数据摘要

   标称属性，给出每个可能取值的频数

   数值属性，给出5数概括及缺失值的个数

- 数据可视化

   使用直方图、盒图等检查数据分布及离群点

##### 数据缺失的处理

- 将缺失部分剔除
- 用最高频率值来填补缺失值
